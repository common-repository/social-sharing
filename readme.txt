Add very easy social share buttons to your website. You can choose from the list of easy social sharing buttons on your site.

== Description ==

A very usefull wp plugin to have in every website. Let your visitors follow and share your website in media with just a click of a button.

== Installation ==

1. Download the plugin Easy Social Sharing from this page and extract it
2. Copy the extracted folder to the "/wp-content/plugins/" directory 
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to Easy Social Sharing select hthe way want to display the social media icons
